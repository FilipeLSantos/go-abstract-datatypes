# Cenarios de simulacao

## Variaveis de entrada

- Distancia (km)
- Regiao: urbana, rural, montanhosa
- Clima: sol, chuva, vento forte

## Meios previstos

- Moto
- Bicicleta
- Drone
- Cavalo

## Casos para demonstracao

- Caso 1: meio opera, calcula tempo e custo.
- Caso 2: meio bloqueado por restricao.
- Caso 3: troca de meio sem mudar o fluxo principal.
